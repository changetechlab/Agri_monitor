# 🌍 Overview

AgriMonitor is an open digital platform developed by **CHANGE TechLab** to support agriculture, climate resilience, and rural development through geospatial intelligence.

The platform combines GIS, AI-assisted workflows, climate monitoring, carbon accounting, and offline-first technologies into a single web application designed for governments, NGOs, researchers, and field teams.
